# Airline Booking System - Production Grade CI/CD

A complete, production-ready airline booking platform built with **Java Spring Boot** backend, **Next.js** frontend, and comprehensive **GitHub Actions CI/CD pipeline**.

## ✨ Quick Start

```bash
# Setup
chmod +x setup.sh
./setup.sh

# Services
Frontend:  http://localhost:3000
Backend:   http://localhost:3001
Swagger:   http://localhost:3001/swagger-ui.html
```

## 🚀 CI/CD Pipeline

5 GitHub Actions workflows for complete automation:
1. Lint & Test (Java + TypeScript)
2. Build Docker Images
3. Deploy to Production
4. Code Quality & Security
5. Documentation & Release
